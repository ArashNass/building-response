# Building Response Lab

**Live:** https://arashnassirpour.com/building-response/

A simplified first-mode dynamic model showing how a building's height,
stiffness, mass, damping and ground-storey condition shape its response to
earthquake shaking. Drive it with real digitized accelerograms (El Centro,
Kobe, Northridge, and others) or synthetic test motions, and watch storey
drift and roof displacement develop in real time.

Built as a single self-contained `index.html`. Hosted on GitHub Pages; any commit
to `main` goes live automatically. Made for learning and exploration, not for
structural design or code compliance.
